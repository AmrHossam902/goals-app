# How to Run the Project

1. **Node.js Version**  
   Both the frontend and backend require **Node.js v20**.

2. **Start MongoDB and Mongo Express**  
   Use Docker Compose to start MongoDb + Express:

   ```bash
   docker compose up -d
   ```

3. **Run Front End**

    - install dependencies
        ```bash
        npm i
        ```
   
    - run the application 
        ```bash
        npm run start
        ```

4. **Run Back End**
    
    - install dependencies
        ```bash
        npm i
        ```
   
    - run the application 
        ```bash
        npm run start
        ```
